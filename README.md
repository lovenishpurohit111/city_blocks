# City Blocks

A modern remake inspired by the classic Nokia City Bloxx game.

## Planned Stack
- Phaser.js
- Vite
- JavaScript
- Deploy on Vercel

## Planned Features
- Classic stacking gameplay
- Increasing difficulty
- Score system
- Mobile responsive controls
- Smooth animations
- Sound effects
- Leaderboard support later

## Goal
Create a polished nostalgia-based arcade stacking game playable on web and Android.
